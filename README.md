<h3 align="center">Aplicação Be The Hero na Semana Omnistack 11 da Rocketseat</h3>

<br />
<p> Desenvolvimento do zero de uma aplicação voltada ao cadastro de ONGS que buscam donativos para casos locais e parceiros que desejarem contribuir com os mesmos.</p>

### **Tecnologias e Bibliotecas utilizadas**

Backend:

- NodeJS (https://github.com/nodejs);
- Knex (https://github.com/knex/knex);
- Sqlite (https://github.com/sqlite/sqlite);
- Jest (https://github.com/facebook/jest);
- Supertest (https://github.com/visionmedia/supertest);

Frontend:

- React (https://github.com/facebook/react);
- React-Native (https://github.com/facebook/react-native);
- Styled-Components (https://github.com/styled-components/styled-components);
- Axios (https://github.com/styled-components/styled-components);
- Expo (https://github.com/expo/expo);
